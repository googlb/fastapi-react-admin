# Technology Stack: FastAPI React Admin

## 1. 概述

本项目是一个用于管理平台的全栈应用，其前端基于 React 和 TypeScript 构建，后端则使用 FastAPI。它结合了现代前端开发的灵活性和高性能后端服务的健壮性。

## 2. 前端技术栈

-   **核心框架:** React 19
-   **构建工具:** Vite 7
-   **路由:** React Router DOM 7
-   **UI 框架:** Ant Design 6
-   **状态管理:** Zustand 5
-   **样式:** Tailwind CSS 4
-   **HTTP 客户端:** Axios
-   **语言:** TypeScript 5

## 3. 后端技术栈 (集成)

-   **核心框架:** FastAPI (作为前端的 API 服务端)

## 4. 开发工具与依赖

-   **代码检查:** ESLint
-   **类型检查:** TypeScript
-   **打包:** Vite
-   **包管理:** pnpm (根据 `pnpm-lock.yaml` 推断)
